# API Testing with Postman

## Overview
This project demonstrates REST API testing using Postman. It includes CRUD operations, response validation, status code verification, and JSON response analysis using the JSONPlaceholder REST API.

## Features
- GET Request Testing
- POST Request Testing
- PUT Request Testing
- PATCH Request Testing
- DELETE Request Testing
- Status Code Validation
- Response Body Validation
- Request & Response Analysis

## Tools Used
- Postman
- REST API
- JSONPlaceholder

## API Used
https://jsonplaceholder.typicode.com/

## Test Scenarios
- Retrieve a single post
- Retrieve all posts
- Create a new post
- Update an existing post
- Partially update a post
- Delete a post
- Verify HTTP status codes
- Validate response body
